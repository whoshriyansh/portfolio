## Run Locally
Clone the project

```bash
  git clone https://github.com/whoshriyansh/portfolio.git portfolio
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run dev
```
